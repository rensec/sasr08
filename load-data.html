Multilevel analysis on network data

3 Load data

For our analysis, we use data on two school classes (id: schoolnr) with 20 and 26 pupils, respectively. We have already created node IDs that are unique across classes, and a pair id that is the same for both dyads (in both directions) within a pair. The data are in long format. “Connected” indicates whether a dyad is connected. We furthermore have the gender of both nodes, and created a variable indicating whether they have the same gender.

# Load data
data <- read_csv("dyads_knecht.csv")
